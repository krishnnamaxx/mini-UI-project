# 🎨 Mini UI Project

A collection of modern UI components built using **React + Tailwind CSS**.
This project focuses on clean design, reusable components, and real-world layout practices.

---

## 🚀 Features

* ⚛️ Built with React (component-based architecture)
* 🎯 Styled using Tailwind CSS
* 🎨 Clean and modern UI design
* 📱 Responsive layout
* ♻️ Reusable components (Cards, Buttons, Layouts)
